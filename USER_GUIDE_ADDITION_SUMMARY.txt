📋 أقسام الدليل (9 أقسام):

┌─────────────────────────────────────────────────────────────────────┐
│ 1. 🔐 تسجيل الدخول للنظام                                         │
├─────────────────────────────────────────────────────────────────────┤
│ • خطوات الدخول خطوة بخطوة                                        │
│ • بيانات الدخول (اسم المستخدم وكلمة المرور)                       │
│ • ملاحظة حول بقاء الجلسة نشطة                                    │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 2. 🍞 إضافة العجنات اليومية                                       │
├─────────────────────────────────────────────────────────────────────┤
│ • خطوات الإضافة (الصباح والمساء)                                 │
│ • معادلة حساب الربح: (عدد العجنات × 10) ÷ 8                      │
│ • أمثلة عملية مع أرقام حقيقية                                    │
│ • كيفية التعديل والحذف                                           │
│ • ملاحظة مهمة: الأرقام بالآلاف                                   │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 3. ⚡ حساب ربح الصمون الكهربائي                                   │
├─────────────────────────────────────────────────────────────────────┤
│ • قسم الإنتاج: الچرك والدائري                                     │
│ • تسعيرة الچرك: 250 دينار للقطعة                                  │
│ • تسعيرة الدائري: 3=500، 4=750، 6=1000                            │
│ • قسم التلف (مهم جداً)                                            │
│ • كيفية حساب صافي الربح بعد خصم التلف                            │
│ • مثال عملي كامل بالأرقام                                        │
│ • ملاحظة: الأرقام بالدينار العادي (وليس بالآلاف)                │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 4. 📉 تسجيل الفروقات                                               │
├─────────────────────────────────────────────────────────────────────┤
│ • شرح مفهوم الفروقات                                              │
│ • أنواع الفروقات: تلف، مطعم، أخرى                                │
│ • خطوات الإضافة                                                  │
│ • معادلة الحساب: (العدد ÷ 8) × 1,000 دينار                       │
│ • أمثلة عملية (تلف، مطاعم، كافتيريا)                             │
│ • البحث والفلترة                                                 │
│ • تنبيه: الفروقات تُخصم من المبلغ الإجمالي                       │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 5. 💰 إدارة المصروفات                                              │
├─────────────────────────────────────────────────────────────────────┤
│ • أنواع المصروفات: عمال، طحين، أخرى                              │
│ • خطوات إضافة مصروف جديد                                         │
│ • مثال على المصروفات اليومية (جدول)                              │
│ • عرض وإدارة المصروفات                                           │
│ • التعديل والحذف                                                 │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 6. 💵 فهم حساب المبلغ الإجمالي النهائي                            │
├─────────────────────────────────────────────────────────────────────┤
│ • المعادلة الكاملة (بتصميم بارز)                                 │
│ • شرح تفصيلي للدخل (ما يُضاف)                                    │
│ • شرح تفصيلي للمصروفات (ما يُخصم)                               │
│ • شرح تفصيلي للفروقات (ما يُخصم)                                │
│ • مثال عملي كامل بالأرقام                                        │
│ • النتيجة النهائية (بتصميم بارز)                                 │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 7. 📚 سجل العجنات والبحث                                          │
├─────────────────────────────────────────────────────────────────────┤
│ • البحث عن سجلات سابقة (يوم واحد أو فترة)                        │
│ • عرض البيانات المُجمعة حسب اليوم                                │
│ • الملخص الشامل لكل يوم                                          │
│ • عرض التفاصيل الكاملة                                           │
│ • التعديل والحذف                                                 │
│ • نصيحة: استخدام السجلات لتحليل الأداء                           │
└─────────────────────────────────────────────────────────────────────┘
