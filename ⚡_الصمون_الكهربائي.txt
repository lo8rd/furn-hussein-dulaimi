التسعيرة حسب الشرائح:

┌──────────┬───────────┐
│ الكمية    │  السعر    │
├──────────┼───────────┤
│ 1 قطعة   │ 250 د     │
│ 3 قطع    │ 500 د     │
│ 4 قطع    │ 750 د     │
│ 6 قطع    │ 1,000 د   │
└──────────┴───────────┘


═══════════════════════════════════════════════════════════════
💡 أمثلة على الحساب التلقائي
═══════════════════════════════════════════════════════════════

مثال 1:
   10 قطع دائري
   = 6 قطع (1,000) + 4 قطع (750)
   = 1,750 دينار ✅

مثال 2:
   15 قطعة دائري
   = 12 + 3
   = (6 × 2) + 3
   = 1,000 + 1,000 + 500
   = 2,500 دينار ✅

مثال 3:
   7 قطع دائري
   = 6 + 1
   = 1,000 + 250
   = 1,250 دينار ✅

مثال 4:
   20 چرك + 10 دائري
   = (20 × 250) + (6 × 1,000 + 4 × 750)
   = 5,000 + 1,750
   = 6,750 دينار ✅

دالة حساب الدائري في db.js:

```javascript
calculateCirclePrice(count) {
    if (count === 0) return 0;
    
    let total = 0;
    let remaining = count;
    
    // تقسيم على شرائح 6
    const sixes = Math.floor(remaining / 6);
    total += sixes * 1000;
    remaining = remaining % 6;
    
    // الباقي
    if (remaining === 4 || remaining === 5) {
        total += 750;
    } else if (remaining === 3) {
        total += 500;
    } else if (remaining === 2) {
        total += 500;
    } else if (remaining === 1) {
        total += 250;
    }
    
    return total;
}
```
