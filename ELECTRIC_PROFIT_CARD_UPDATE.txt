═══════════════════════════════════════════════════════════════════════
✅ التحديث:
═══════════════════════════════════════════════════════════════════════

تم إضافة بطاقة جديدة في الصفحة الرئيسية (index.html) تعرض:

📊 **⚡ ربح الكهربائي**
   • يعرض صافي ربح الكهربائي (بعد خصم التلف)
   • بالدينار العادي (ليس بالآلاف)
   • خلفية بنفسجية مميزة (gradient)
   • أيقونة ⚡ للتمييز


═══════════════════════════════════════════════════════════════════════
📋 البطاقات في الصفحة الرئيسية (6 بطاقات):
═══════════════════════════════════════════════════════════════════════

1. **عجنات اليوم**
   - عدد العجنات
   - خلفية عادية

2. **ربح العجنات**
   - ربح العجنات فقط
   - بالألف دينار
   - خلفية عادية

3. **⚡ ربح الكهربائي** ← جديد!
   - صافي ربح الكهربائي (بعد خصم التلف)
   - بالدينار العادي
   - خلفية بنفسجية (gradient: #667eea → #764ba2)

4. **💰 المبلغ الإجمالي**
   - ربح العجنات + ربح الكهربائي
   - بالدينار العادي
   - إطار ذهبي

5. **مصاريف اليوم**
   - إجمالي المصاريف
   - بالدينار العادي
   - خلفية عادية

6. **صافي الربح**
   - الربح بعد خصم المصاريف والفروقات
   - بالألف دينار
   - خلفية عادية

🎨 التصميم:
═══════════════════════════════════════════════════════════════════════

✓ البطاقة الجديدة:
  - الخلفية: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
  - النص: أبيض
  - الحجم: 2rem (أصغر قليلاً من الإجمالي)
  - الأيقونة: ⚡

✓ التخطيط:
  - grid-template-columns: repeat(auto-fit, minmax(280px, 1fr))
  - يتكيف تلقائياً مع حجم الشاشة
  - 3 بطاقات في السطر على الشاشات الكبيرة
  - 2 أو 1 على الشاشات الصغيرة

📊 البيانات المعروضة:
═══════════════════════════════════════════════════════════════════════

**ربح الكهربائي** = صافي ربح الكهربائي (net_profit)
- يُجلب من stats.electricProfit
- يتم حسابه في db.getStats() كالتالي:

```javascript
const totalElectricProfit = electric.reduce((sum, e) => {
    const profit = parseFloat(e.net_profit || e.total_profit || 0);
    return sum + profit;
}, 0);
```

**المبلغ الإجمالي** = (ربح العجنات × 1000) + ربح الكهربائي
```javascript
const totalProfitWithElectric = (totalProfit * 1000) + totalElectricProfit;
```

📝 مثال عملي:
═══════════════════════════════════════════════════════════════════════

الحالة: يوم عمل عادي

**المدخلات:**
- ربح العجنات: 150 ألف دينار (= 150,000 دينار)
- ربح الكهربائي الإجمالي: 40,000 دينار
- تلف الكهربائي: 6,000 دينار
- صافي ربح الكهربائي: 40,000 - 6,000 = 34,000 دينار

**العرض في الصفحة الرئيسية:**

1. عجنات اليوم: 95 عجنة
2. ربح العجنات: 150 ألف دينار
3. ⚡ ربح الكهربائي: 34,000 دينار ← البطاقة الجديدة
4. 💰 المبلغ الإجمالي: 184,000 دينار (150,000 + 34,000)
5. مصاريف اليوم: 45,000 دينار
6. صافي الربح: 139 ألف دينار
